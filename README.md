# OOD Detection Framework
Code for our paper **Confidence-based Out-of-Distribution Detection: A Comparative Study and Analysis**

Implemented methods: 
* Maximum Class Probability
* Deep Ensembles
* ODIN
* Mahalanobis Distance
* Center Loss 
* Mahalanobis Ensemble
* Monte Carlo Dropout
